# Login Responsivo

Projeto de interface de login desenvolvido com HTML e CSS, com foco em responsividade e layout moderno.

## 🔗 Acesse o projeto
https://tst-responsive-login.netlify.app/

## 💻 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 📌 Funcionalidades
- Layout responsivo (adaptado para mobile e desktop)
- Interface de login moderna
- Organização de estrutura com HTML semântico
- Estilização com CSS

## 📁 Como executar
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
